Lembrete de tags HTML(h1, h2, ... h6)

Um parágrafo simples (p)

Usando (br)
para quebrar a linha

Símbolos de maior e menor: > (> + ;), < (< + ;)

Adicionando emojis: 😀 &# + numeração hexadecimal + ;

Adiconar imagens: tag (img src="" alt=""). Caso necessário, use ../ para voltar uma pasta para então acessar o local correto do arquivo

Modificadores de texto:
(strong) ou (b) para negrito
(em) ou (i) para itálico
(mark) para marcar um texto
(del) para um texto excluído
(ins) para um texto inserido
(sup) para um texto sobrescrito
(sub) para um texto subscrito
(code) deixa o texto como código
(q) para citações simples
(blockquote) para citações

completas

(pre) mantém o formato no site em um 
   código

(abbr) para adicionar abreviações ao colocar o mouse na palavra

Listas:
(ol) cria listas ordenadas, (ul) cria listas não ordenadas; (li) cria os elementos da lista
Lista ordenada:

  1. type="1" start="x"
  2. type="A" start="x"
  3. type="a" start="x"
  4. type="I" start="x"
  5. type="i" start="x"
Lista não ordenada:
  • type="square"
  • type="disc"
  • type="circle"

Lista de definição (dl):
(dt) Definition term
(dd) definition description

Links:
(a href="https://") para acessar um link externo
(a href="nome-arquivo") para trocar de página no site
(a href="pasta/nome-download.formato") para baixar o arquivo desejado
